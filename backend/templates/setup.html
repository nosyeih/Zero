Configuración Inicial

Conecta tu Google Sheet para habilitar Finanzas y Rentabilidad.

  1. Copia la hoja de cálculo: Abre esta plantilla oficial.
    Una vez abierta, ve a Archivo > Hacer una copia para guardarla en tu Google Drive personal.
    Asegúrate de tener las siguientes columnas en la primera fila de la hoja 'Transacciones': FECHA_INGRESO, EMPRESA, CONCEPTO_PAGO, MONEDA, MONTO, NRO_OPERACION, DETALLE
  2. Abre el editor de Apps Script: En tu hoja de cálculo, ve a Extensiones > Apps Script.
  3. Pega el código: Borra cualquier código existente y pega lo siguiente (actualizado para Rentabilidad):
  4. Implementar como aplicación web: Haz clic en Implementar > Nueva implementación.
    • Selecciona el tipo: Aplicación web.
    • Descripción: "API Finanzas v2".
    • Ejecutar como: Yo (tú mismo).
    • Quién tiene acceso: Cualquier usuario.
    Haz clic en Implementar.
    ¡Importante! Si actualizas el código, debes crear una Nueva implementación (New Deployment) para que los cambios surtan efecto.
